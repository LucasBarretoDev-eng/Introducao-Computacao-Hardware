
# 💻 Introdução à Computação

A disciplina de **Introdução à Computação** é o ponto de partida fundamental no curso de Engenharia de Software. Ela funciona como a base teórica e prática necessária para entender como a tecnologia é construída e como os sistemas operam desde os níveis mais baixos.

---

## 📚 O Que se Estuda?
O objetivo principal é oferecer uma visão panorâmica do universo da computação. Os tópicos geralmente incluem:

* **História e Evolução:** Desde os primeiros computadores mecânicos até a computação em nuvem e quântica.
* **Arquitetura de Computadores:** Como o hardware (CPU, Memória, Entrada/Saída) se comunica.
* **Sistemas de Numeração:** O entendimento de como o computador "processa" dados em binário, octal e hexadecimal.
* **Lógica de Programação:** Introdução ao pensamento algorítmico e à resolução estruturada de problemas.
* **Sistemas Operacionais:** O papel do software que gerencia o hardware e provê a interface para o usuário.

---

## 🛠️ O Papel na Engenharia de Software
Diferente de um curso estritamente técnico, na Engenharia de Software esta disciplina foca em pilares estratégicos:

1.  **Abstração:** Aprender a isolar a complexidade de um problema para focar no que é essencial.
2.  **Raciocínio Lógico:** Desenvolver a base cognitiva para aprender qualquer linguagem de programação futuramente.
3.  **Ética e Profissionalismo:** Discussões fundamentais sobre o impacto da tecnologia na sociedade e a responsabilidade do engenheiro.

> **Nota:** Esta matéria transforma o estudante de um usuário de tecnologia em um criador de soluções, fornecendo o "vocabulário" técnico necessário para toda a graduação.
